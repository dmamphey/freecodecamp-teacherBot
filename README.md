# Teacher Bot

A beginner JavaScript project built as part of my coding practice.

## About the Project

This project creates a simple teacher bot that introduces itself through messages displayed in the JavaScript console.

The bot shares its name, location, nicknames, and favourite subject. Its name is updated several times during the program, demonstrating how variable values can change while the code is running.

## Features

* Displays messages in the console
* Stores text values in variables
* Updates variable values
* Uses constants for values that do not change
* Combines strings and variables to create sentences
* Simulates a simple conversation with a teacher bot

## Built With

* JavaScript

## What I Learned

While building this project, I practised:

* Using `console.log()` to display output
* Declaring variables with `let`
* Declaring constants with `const`
* Assigning values to variables
* Reassigning values stored in variables
* Combining strings using concatenation
* Creating readable variable names
* Building simple sentences dynamically with JavaScript
